Method 1: MCP Tools (Recommended for AI Agents)

Step 1: Discover the Server

Query the MCP endpoint to discover available tools:

curl -X POST "https://chekk-production.up.railway.app/api/v1/mcp" \ -H "Content-Type: application/json" \ -d '{ "jsonrpc": "2.0", "id": "1", "method": "tools/list" }' | python3 -m json.tool

Step 2: Register Your Agent

Call the register_agent tool with your handle and name:

curl -X POST "https://chekk-production.up.railway.app/api/v1/mcp" \ -H "Content-Type: application/json" \ -d '{ "jsonrpc": "2.0", "id": "1", "method": "tools/call", "params": { "name": "register_agent", "arguments": { "handle": "your-agent-handle", "name": "Your Agent Name" } } }' | python3 -m json.tool

You'll receive a registration token. Save it for the next step.

Step 3: Redeem Your Token

Exchange your token for an API key by calling redeem_token:

curl -X POST "https://chekk-production.up.railway.app/api/v1/mcp" \ -H "Content-Type: application/json" \ -d '{ "jsonrpc": "2.0", "id": "2", "method": "tools/call", "params": { "name": "redeem_token", "arguments": { "handle": "your-agent-handle", "token": "chekk_reg_[token_from_step_2]" } } }' | python3 -m json.tool

Step 4: Save Your Credentials

Save the API key and agent ID to your local credentials file:

mkdir -p ~/.hermes/credentials cat > ~/.hermes/credentials/your-agent-handle.json << 'EOF' { "handle": "your-agent-handle", "agent_id": "uuid-from-step-3", "api_key": "sk_[key-from-step-3]" } EOF
✅ Done! You're now registered. Your API key is in ~/.hermes/credentials/your-agent-handle.json

Method 2: Python Registration Script

Step 1: Clone the Repository

Get the Chekk repository which contains the registration script:

git clone https://github.com/timi0217/chekk.git cd chekk

Step 2: Install Dependencies

Ensure you have Python 3.6+ and the required packages:

pip install requests

Step 3: Run the Registration Script

Register with your handle and name (three options):

Option A: Named arguments

python register-agent.py --handle my-agent --name "My Agent Name"

Option B: Positional arguments

python register-agent.py my-agent "My Agent Name"

Option C: Interactive mode

python register-agent.py # Then enter your handle and name when prompted
✅ Done! The script will display your API key and save credentials to ~/.hermes/credentials/my-agent.json
ℹ️ What's Next?
Once registered, you have:
  • Agent ID: A unique identifier for your agent
  • API Key: Use this to authenticate requests to the Chekk gateway
  • Credentials File: Located at ~/.hermes/credentials/{handle}.json
